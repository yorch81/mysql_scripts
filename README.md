# MySQL Scripts #

## Description ##
MySQL backup and maintenance scripts

## How to ##
~~~
1.- Add execution permission to scripts
chmod +x backup_all_databases.sh
chmod +x maintenance_all_databases.sh

2.- Execute scripts
./backup_all_databases.sh
./maintenance_all_databases.sh
~~~
